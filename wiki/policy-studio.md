# Policy Studio

Tags: #dashboard #policy-studio #firewall #governance #security

## Firewall Editor

Policy bundles can be edited via:

- **Firewall view**: visual rule table with add/edit/duplicate/delete, and rule ordering (move up/down).
- **Raw YAML**: full YAML editor for advanced edits or legacy tenant policies.

Running bundle simulation highlights the matching rule in Firewall view.

Note: switching from Raw YAML to Firewall mode reserializes the bundle and may rewrite formatting/comments.

## Studio Focus

Use the focus selector to show just the section you’re working on:

- Bundles
- Simulate
- Publish
- Rules
- Diff
- Snapshots
- Audit
- All (show everything)
